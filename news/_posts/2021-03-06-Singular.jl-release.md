### Summary of changes

The main change in this release is that it enables several Singular extensions which were not available before. Specifically, these are now enabled (if you need others, let us know)
- gfanlib
- syzextra
- customstd
- interval
- subsets
- loctriv
- gitfan
- freealgebra
